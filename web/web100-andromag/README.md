# WEB 100 - Andromag

## Задача
Получить пароль от базы данных. Дан доступ на сайт и исходник(index.php) без паролей.

## Легенда
Наши разведчики уверены, что в андромаге на планете Мрачных торговцев продают запрещенные компоненты для изготовления черных дыр. Необходимо получить лазерное клеймо от их инфосетей. В случае подтверждения информации мы будем вправе атаковать их.

## Условия на сервере

- Сервисы: PHP, MYSQL, APACHE
- Порт APACHE: 80
- Разрешения пользователя MYSQL: SELECT ON `table_1_1`

### Флаг: kctfF1R5tb100d

## Решение:
Фишка в кривой реализации функции compact, результат которой передается в SQL запрос. Можно заметить, что при отправке в GET параметрах невалидных данных выпадают исключения базы данных. А так же, что GET параметр "order" передается в функцию compact. Таким образом в GET параметр "order" закидываем "mysql_passwd", а в "page" закидываем нуль или любое отрицательное число для вызова исключения и видим в нем флаг.

## Развертывание

Дамп базы данных дан в файле **data.sql**
